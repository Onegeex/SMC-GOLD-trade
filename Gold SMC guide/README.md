# SMC Gold Guide

A trading guide for XAUUSD (Gold) using Smart Money Concepts (SMC) on TradingView.

## Overview

`SMC_Gold_Guide.html` is a styled guide that explains the SMC trading setup for Gold on the H1 timeframe. It covers:

- Bullish and bearish order blocks
- Fair value gaps (FVG)
- Liquidity sweeps and stop-hunt setups
- London and New York kill zones
- Pro buy and sell entry rules
- TradingView installation steps

## What is included

- `SMC_Gold_Guide.html` — visual guide with chart concepts, trade setup rules, session timing, and install instructions
- `README.md` — project summary and usage notes

## How to use

1. Open `SMC_Gold_Guide.html` in a browser.
2. Review the SMC concepts and trade setup rules.
3. Use the guide alongside TradingView and Pine Script for XAUUSD H1.

## Installation notes

The guide assumes you are using TradingView and a Pine Script indicator for SMC on XAUUSD.

- Set chart timeframe to `1H`
- Use UTC timezone for correct kill zone shading
- Confirm bias on higher timeframes (H4/D1) before trading H1 signals
